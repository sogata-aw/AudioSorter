# AudioSorter

## Fonctionnalités

Dans cette application en PyQt, il est possible de donner des fichiers et de les trier par 
artistes ou par album dans un dossier de sortie.

## Utilisation

1. Choisissez un dossier de sortie (c'est dans celui-ci que seront créés les dossiers et où 
apparaitront vos fichiers)
2. Faites glisser les fichiers dans l'application
3. Cliquer sur trier par artiste ou par album

Si vous triez par artistes, vous pouvez choisir d'avoir uniquement le nom du premier artiste 
ou de prendre tous les noms d'artistes (d'autres paramètres sont à venir)
